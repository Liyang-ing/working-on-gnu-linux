#### 將內容轉成 HTML

這指令會產生個與 Vim 上看起來一樣的 .html，在 7.3 的版本更會將 CSS 獨立出來，並加入部落格、網站上的 CSS 裡，可事半功倍。

這對想把程式碼放到網頁上的人而言是非常好的功能，它會，

open current file to browser with F5
